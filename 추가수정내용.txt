수정09 — 버튼 활성화 엔진 (기본)

퍼센트 버튼 녹색 활성화
매수예약/즉시매수/즉시매도 녹색 테두리 활성화
손절/슬리피지/실행 버튼 활성화 (기본)

수정10 — 버튼 활성화 강화

손절 버튼 → 강렬한 빨간 네온 + 깜빡임
실행 버튼 → 강렬한 녹색 네온 + 깜빡임

수정11·12·13 — 버튼 동작 로직 완성

퍼센트 버튼 → 다른 버튼/입력창 클릭 시 자동 비활성화
매수예약 ↔ 금액 입력창 연동 활성화 (파란 네온)
즉시매수 → 강렬한 빨간 네온 + 깜빡임, 토글 방식
즉시매도 → 강렬한 주황 네온 + 깜빡임, 토글 방식
즉시매수/즉시매도 클릭 시 1행·2행 모든 버튼 비활성화
손절 비활성화 시 2행 전체 비활성화
